A powerful reservation and bookings plugin to book your restaurant/cafeteria table, seat bookings, and appointment bookings. It is an all in one reservation solution plugin with the features of weekly hours management, holiday settings, and much more.

== Description ==

The All in one reservation plugin is a free solution for everyone who is looking for a reservation system on their website. All is one reservation plugin provides you with a variety of settings, calendar design, time slots, and much more for free.

**Live Demo:** [aior.solwininfotech.com](https://aior.solwininfotech.com/)

**Documentation:** [All in one reservation Documentation Link](https://solwininfotech.com/documents/wordpress/all-in-one-reservation/)

= 7+ Different Calendars and Forms =
The plugin offers 7+ different and unique calendars and forms to make your appointment booking page more attractive.

= Flexible reservation Forms =
Add any type of fields in the appointment form like textbox, textarea, select box, checkbox, etc... So any type of form for any type of reservation can be created.

= Multiple reservation Forms =
Create unlimited reservation forms for your website.

= Shortcodes Support =
Using the shortcode, you can use it anywhere you need, place it on a page, or posts it wherever you want.

= Confirm reservation =
From the backend, Admin confirms the reservation, and customers get the notification when the reservation is confirmed.

= Cancel reservation =
Reservations can be canceled and customers will receive a notification of the cancellation.

= Fully Customizable Admin Interface =
The plugin provides an easy-to-use and user-friendly admin interface.

= Multiple Browser Compatibility =
The plugin is compatible with multiple web browsers, like Firefox, Chrome, Safari, and Opera.

= Import & Export Layout =
Users can easily import reservation forms and export appointment lists.

= Multilingual Translation Ready =
All in One Reservation is ready to use with any language.

= Technical Support =
We're active for any support issues and feature suggestions. So hope you will love it. Please contact us at [support.solwininfotech.com](http://support.solwininfotech.com)

== Installation ==

1. Upload the **all-in-one-reservation.zip** file Via WordPress Admin > Plugins > Add New,
2. Alternately, upload the **all-in-one-reservation** folder to the /wp-content/plugins/ directory via FTP,
3. Activate the **All in One Reservation** plugin from Admin > Plugins.
